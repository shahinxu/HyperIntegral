# HyperPINN

This repository contains two versions of HyperPINN implementations:

## HyperPINN_Drive
The driving version of HyperPINN implementation.

## HyperPINN
The main HyperPINN implementation with various examples including:
- Rossler Oscillators
- Lotka-Volterra
- Driving systems

Both implementations use Physics-Informed Neural Networks (PINNs) for discovering hypergraph structures in dynamical systems.
